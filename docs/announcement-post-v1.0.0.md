# SmartCalculator v1.0.0 — Announcement Post

---

## English

**SmartCalculator v1.0.0 is now available.**

A premium handheld-style scientific calculator for Windows Desktop. Built with WPF and .NET 8.

### Features
- 9 calculation modes: COMP, BASE-N, STAT, CMPLX, MATRIX, VECTOR, TABLE, EQN/SOLVE
- Full scientific function set with SHIFT/ALPHA key system
- Dark AMOLED-style UI designed for focus
- Offline, portable, calculator-only — no data, no ads, no internet

### Modes at a Glance
| Mode | What it does |
|------|-------------|
| BASE-N | Decimal/Hex/Binary/Octal with bitwise operations |
| STAT | Mean, standard deviation, min/max on your data |
| CMPLX | Complex number arithmetic |
| MATRIX | 2x2 and 3x3 matrix calculations |
| VECTOR | 2D/3D vector operations including dot and cross products |
| TABLE | Generate function tables with start/end/step |
| SOLVE | Numeric equation solving via solve() |

### Download
- **Setup installer**: `SmartCalendar_Setup_v1.0.0.exe` (recommended for beginners)
- **Portable ZIP**: Extract and run, no installation needed
- **Framework-dependent**: Smaller download, requires .NET 8

All packages available on GitHub.

## العربية

**يتوفر الآن الإصدار النهائي من SmartCalculator v1.0.0.**

آلة حاسبة علمية متميزة على نمط الآلات الحاسبة المحمولة لنظام Windows. بُنيت باستخدام WPF و .NET 8.

### الميزات
- 9 أوضاع حسابية: COMP، BASE-N، STAT، CMPLX، MATRIX، VECTOR، TABLE، EQN/SOLVE
- مجموعة كاملة من الدوال العلمية مع نظام مفتاحي SHIFT/ALPHA
- واجهة داكنة بتصميم AMOLED للتركيز
- بدون اتصال بالإنترنت، محمولة، آلة حاسبة فقط — لا بيانات ولا إعلانات

### الأوضاع باختصار
| الوضع | الوظيفة |
|-------|---------|
| BASE-N | التحويل بين الأنظمة العددية مع العمليات المنطقية |
| STAT | الوسط الحسابي والانحراف المعياري للبيانات |
| CMPLX | العمليات الحسابية على الأعداد المركبة |
| MATRIX | عمليات المصفوفات 2×2 و 3×3 |
| VECTOR | عمليات المتجهات ثنائية وثلاثية الأبعاد |
| TABLE | إنشاء جداول الدوال الرياضية |
| SOLVE | حل المعادلات العددية باستخدام solve() |

### التحميل
- **برنامج تثبيت**: `SmartCalculator_Setup_v1.0.0.exe` (مُوصى به للمبتدئين)
- **نسخة محمولة ZIP**: فك الضغط وشغّل، لا يحتاج تثبيت
- **نسخة تعتمد على .NET 8**: تحميل أصغر، تحتاج تشغيل .NET

جميع الحزم متوفرة على GitHub.
